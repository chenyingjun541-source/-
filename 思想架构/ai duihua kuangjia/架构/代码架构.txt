## 三、热插拔插槽 (hot-swap.ts)

```typescript
/**
 * 热插拔插槽接口 - houduan
 * 这是你的代码与 Airi 之间的"插槽"
 */

export interface HotSwapMessage {
  id: string
  type: 'user' | 'ai'
  content: string
  timestamp: number
}

export interface HotSwapResult {
  content: string
  modified: boolean
  metadata?: Record<string, any>
}

// 核心导出函数
export async function onMessage(message: HotSwapMessage): Promise<HotSwapResult>
export async function onInit(): Promise<void>
export async function onDestroy(): Promise<void>
export async function onConversationStart(conversationId: string): Promise<void>
export async function onConversationEnd(conversationId: string): Promise<void>
```

**使用方式**：
| 模式 | 是否加载 |
|------|----------|
| 直连模式 (direct.bat) | ❌ 不加载 |
| 代理模式 (proxy.bat) | ✅ 自动加载 |

## 四、智能路由架构 (local.model/)

```
local.model/
├── router.ts          # 核心路由引擎
│   ├── route()        # 主路由方法
│   ├── keywordMatch()  # 关键词匹配
│   ├── learnedMatch()  # 学习词库匹配
│   └── modelJudge()    # 本地模型判断
│
├── keyword-db.ts      # 关键词数据库
│   └── matchKeyword() # 毫秒级匹配
│
├── learned-db.ts      # 学习词库
│   ├── matchLearned()  # 已学习短语匹配
│   ├── learnPhrase()   # 学习新短语
│   └── promoteToKeyword() # 提升为关键词
│
├── local-judge.ts     # 本地模型判断器
│   ├── start()        # 启动后台判断
│   ├── stop()         # 停止
│   └── scorePhrase()  # 评分短语复杂度
│
└── index.ts           # 统一导出
```

**路由决策流程**：
```
用户消息
    │
    ▼
┌─────────────────────────────────────────────────────┐

│                   作为路由目标                       │
└─────────────────────────────────────────────────────┘
```

## 七、关键代码流程

### 7.1 消息处理流程

```typescript
// 1. 代理收到消息
proxy.on('message', handleMessage)

// 2. 调用热插拔钩子
const result = await onMessage(message)

// 3. 智能路由决策
const routeResult = await smartRouter.route(message)

// 4. 根据路由目标分发
switch (routeResult.target) {
  case 'local':
    return await callLocalModel(message)      // 本地模型
  case 'cloud-quick':
    return await callCloudQuick(message)      // 云端快速
  case 'cloud-deep':
    return await callCloudDeep(message)       // 云端深度
}

// 5. 结果处理
if (routeResult.target === 'cloud-deep') {
  await refineAndArchive(response)  // 深度处理后存档
}
```

### 7.2 记忆更新流程

```
短期记忆 (对话中)
    │ 达到阈值
    ▼
中期记忆 (话题摘要) ──→ 云端提炼 ──→ 存档
    │                            │
    │                            ▼
    │                     背景文件 (精简版)
    │                            │
    │                            ▼
    │                     长期记忆 (大脑)
    │                            │
    ◄────────────────────────────┘
```
