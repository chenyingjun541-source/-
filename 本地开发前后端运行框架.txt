# 本地开发前后端运行框架

## 一、启动模式总览

| 模式 | 启动方式 | 前端连接 | 热插拔 | 适用场景 |
|------|----------|----------|--------|----------|
| **直连模式** | `direct.bat` | 直接连接6112 | ❌ 不加载 | 调试airi原生功能 |
| **代理模式** 🔴 | `proxy.bat` | 连接5137 | ✅ 加载 | 启用houduan自定义逻辑 |


## 四、热插拔插槽 (hot-swap.ts)

### 4.1 接口定义

```typescript
// 消息结构
interface HotSwapMessage {
  id: string
  type: 'user' | 'ai'
  content: string
  timestamp: number
}

// 结果结构
interface HotSwapResult {
  content: string
  modified: boolean
  metadata?: Record<string, any>
}

// 核心导出函数
export async function onMessage(message: HotSwapMessage): Promise<HotSwapResult>
export async function onInit(): Promise<void>
export async function onDestroy(): Promise<void>
export async function onConversationStart(conversationId: string): Promise<void>
export async function onConversationEnd(conversationId: string): Promise<void>
```

### 4.2 消息处理流程

```typescript
// proxy/index.ts 中的处理流程

// 1. 收到用户消息
app.post('/api/chat', async (req, res) => {
  const message: HotSwapMessage = {
    id: generateId(),
    type: 'user',
    content: req.body.message,
    timestamp: Date.now()
  }

  // 2. 调用热插拔钩子
  const result = await onMessage(message)

  // 3. 获取路由结果
  const routeResult = result.metadata?.route

  // 4. 根据路由分发到不同模型
  if (routeResult?.target === 'local') {
    // 本地模型处理
  } else if (routeResult?.target === 'cloud-quick') {
    // 云端快速处理
  } else if (routeResult?.target === 'cloud-deep') {
    // 云端深度处理 + 存档
  }

  // 5. 返回结果
  res.json({ response: result.content })
})
```

## 五、智能路由 (smartRouter)

### 5.1 路由决策流程

```
用户消息
    │
    ▼
┌─────────────────────────────────────────────────────┐
│              三方并行竞速                           │
│                                                      │
│  ┌──────────────┐  ┌──────────────┐  ┌────────────┐  │
│  │ 关键词匹配   │  │ 学习词库    │  │ 本地模型   │  │
│  │ (10ms)     │  │ (10ms)     │  │ (3秒超时)  │  │
│  └──────┬──────┘  └──────┬──────┘  └─────┬──────┘  │
│         │                │                │          │
│         └────────────────┼────────────────┘          │
│                          ▼                          │
│                   选举胜出者                         │
│                                                  │
│  优先级：关键词 > 学习词 > 本地模型 > 默认           │
└─────────────────────────────────────────────────────┘
    │
    ├── local ────→ 本地模型 (Qwen-1.8B) <2秒响应
    ├── cloud-quick ──→ 云端快速 (百炼) <5秒响应
    └── cloud-deep ──→ 云端深度 (DeepSeek) 质量优先
```

### 5.2 路由目标说明

| 目标 | 处理方式 | 响应时间 | 适用场景 |
|------|----------|----------|----------|
| `local` | Ollama本地模型 | <2秒 | 简单问候、脱敏润色 |
| `cloud-quick` | 阿里云百炼 | <5秒 | 日常对话、情感回应 |
| `cloud-deep` | DeepSeek | 不限时 | 记忆提炼、复杂推理 |
